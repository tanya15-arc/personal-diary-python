# 📖 Personal Diary (Python)

A simple command-line Personal Diary application built with Python. It allows users to write daily thoughts, read previous entries, delete all diary entries, and exit the program through a menu-driven interface.

## ✨ Features

- 📝 Add new diary entries
- 📚 Read all saved entries
- 🗑️ Delete all diary entries
- 💾 Automatically saves entries in a text file (`diary.txt`)
- 🎯 Simple and beginner-friendly menu interface

## 🛠️ Technologies Used

- Python 3
- File Handling
- Loops
- Conditional Statements
- Exception Handling

## 📂 Project Structure

```
personal-diary-python/
│── Diary.py
│── diary.txt (generated automatically)
│── README.md
```

## 🚀 How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

```bash
python Diary.py
```
